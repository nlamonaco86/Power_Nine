power Nine began as a hobby project and learning exercise in DOM maniuplation and AJAX calls, utiliziting Bootstrap, jQuery, and the Scryfall API to search for and display images of Magic: The Gathering cards.

After the initial version was complete, I began to learn more about other, more detailed technologies that could transform my page into something far more detailed and interesting. Utilizing Node.js, Express MySQL, and Sequelize, I was able to now build out a fully-functional server and database structure with which the end user and the Scryfall API could both interact with.

This allows users the ability to save their collections and cards to the database, and with a functional user authentication system in Passport, keep their information and account secure. Users can then view the sets owned by other users, and request a trade.

Although power Nine is themed around Magic: The Gathering and its associated characters appear as the users, no knowledge of the actual game is required for this demonstration.

The application is intended as a demonstration of my understanding of the associated technologies and their uses, and the concepts are displayed in action here. I believe an important part of learning is knowledge transferrence, and I am able to adapt what has been done here to apply in a number of different scenarios that are commonly found in real-world settings.
